Update Font Directories/FontFiles:

Run the cross-platform installer from this folder:

python update_fonts.py --font FROMVS.ttf

If the font lives in a different source folder, pass it explicitly:

python update_fonts.py --font FROMVS.ttf --source-dir "C:/path/to/training/output/fonts"

The script installs to the current user's font locations on Windows, macOS, and Linux,
and refreshes the Linux font cache when available.
