# BiblionOCR Intro Video Workspace

This workspace holds the production materials for the first BiblionOCR introduction video.

The public release of the introduction video is available here:

https://www.youtube.com/watch?v=zrJQzivQwT4

It is intentionally separated from application runtime code and Developer Services.
Use the existing licensed image categories in [Developer/assets](c:/Users/Max/Projects/BiblionOCR/Developer/assets) as source material without moving or renaming those folders.

## Folder Map

- `audio/`: narration, music, sound design, and final mixed stems.
- `exports/`: rendered previews, review cuts, and final delivery files.
- `project/`: editable files from the chosen video editor, motion tool, or compositing tool.
- `scripts/`: voiceover text, shot lists, timing notes, and production drafts.
- `storyboards/`: boards, frame plans, animatic references, and asset selection notes.

## Usage Notes

- Keep raw licensed imagery in `Developer/assets/`.
- Reference storyboard categories by their current A-E folder names.
- Store only video-production materials in this workspace.
